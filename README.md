## \
**# Installation**

### Prérequis

Pour utiliser le projet il vous faudra au minimum :

* NPM
* Node

### Initialiser le projet

```
npm init
npm install discord.js @discordjs/rest discord-api-types @discordjs/voice @discordjs/builders
```
